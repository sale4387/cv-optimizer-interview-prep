## TASK-009 — Define Section Revision Models and Validation Rules

### Status

Backlog

### Goal

This task will allow user to select the sections that need revision and write comments to it so the new AI call can be made.

### Description

In this task a logic will be developed which will send another call to AI asking to write selected sections again. After results are in, revised sections will replace sections that needed review and another round of cleaning andavalidation will be done.

- [ ] When CV is ready it gets saved to db as record with status draft
- [ ] when user opens preview two buttons are displayed - request changes and accept cv
- [ ] Clicking accpet changes status in db and allows savng as pdf from print feature
